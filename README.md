<h3> How to install: </h3>

cd ~  </br>
git clone git://github.com/AliceYuan/vim-config-AY.git .vim  </br>
cd ~/.vim  </br>
ln -s ~/.vim/.vimrc ~/.vimrc  </br>
git submodule init  </br>
git submodule sync  </br>
